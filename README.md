# EyeCue Video Gallery

This repository contains the source code for the EyeCue Video Gallery, hosted on GitHub Pages. The gallery showcases videos generated by **EyeCue**, a multi-model AI agent developed by Aganitha AI that interprets molecular dynamics (MD) trajectories and provides audio-visual explanations.

## Local Development (macOS)

To build and test this Jekyll site locally on a Mac, follow these instructions:

### Prerequisites

1. **Xcode Command Line Tools**: Ensure you have the Apple command line tools installed. Open your terminal and run:
   ```bash
   xcode-select --install
   ```

2. **Ruby**: While macOS comes with a pre-installed version of Ruby, it is strongly recommended to use a package manager like [Homebrew](https://brew.sh/) to install a separate Ruby environment. This avoids permission issues with the system Ruby.
   ```bash
   # Install rbenv and ruby-build using Homebrew
   brew install rbenv ruby-build
   
   # Set up rbenv in your shell (follow the instructions printed by this command)
   rbenv init
   
   # Install a recent Ruby version and set it as the global default
   rbenv install 3.4.9
   rbenv global 3.4.9
   ```

3. **Bundler**: With your user-level Ruby set up, install Bundler to manage the project's Ruby gems:
   ```bash
   gem install bundler
   ```

### Installation

1. Clone this repository to your local machine.
2. Navigate to the repository directory in your terminal.
3. Install the required Jekyll dependencies using Bundler:
   ```bash
   bundle install
   ```

### Running the Site Locally

1. Once the dependencies are installed, start the local Jekyll development server:
   ```bash
   bundle exec jekyll serve --livereload
   ```
   *(The `--livereload` flag automatically refreshes your browser when you make changes to files.)*
2. Open your web browser and navigate to `http://localhost:4000` to view the site.
